# macroknob-shield
A simple 2-key, 1 rotary encoder shield for ZMK. Designed to complement any ZMK keyboard as a dongle.

The firmware can be as-is to be used standalone.

To use as a dongle for an existing keyboard, follow the guide below and the instructions in each file to integrate the configuration from your existing keyboard into this. The 

As a high level overview, you will build your existing keyboard's firmware as per usual, with a slight modificatio to the build options, while you will build the MacroKnob's firmware separately using this repository as a base.

The 'lily58' branch is an example for the SplitKB Lily58. It is the configuration I use for my own daily driver.
